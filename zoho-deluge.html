Course Syllabus Overview

This is a brief overview of the syllabus. For more detailed information on each topic, please download the full syllabus.

Download Full Syllabus

Week 1: Introduction to Deluge

  • Overview of Deluge scripting language and use cases in Zoho products
  • Setting up the environment (Zoho Creator, Zoho CRM, etc.)
  • Basic syntax and structure
  • Variables, data types (strings, numbers, boolean)
  • Comments and best practices

Week 2: Operators and Control Structures

  • Arithmetic, comparison, and logical operators
  • Conditional statements (if, else, switch)
  • Looping structures (for, while)
  • Working with collections: lists, maps, key-value pairs
  • Understanding function definitions and calls

Week 3: Functions and Date Handling

  • Built-in functions (string manipulation, math functions)
  • Creating custom functions
  • Handling date and time in Deluge
  • Formatting and manipulating date values
  • Review and practice with mini projects

Week 4: Intermediate Deluge Concepts

  • Integration with Zoho applications and APIs
  • Fetching and modifying records in Zoho apps
  • Implementing input forms and validating user inputs
  • Functions in reports and workflows: triggers and execution context
  • Error handling and debugging techniques

Week 5: Building Workflows and Automation

  • Building workflows for automation
  • Understanding workflow triggers and execution context
  • Building custom workflows for different use cases
  • Review and practice with workflow creation
  • Refining mini projects with workflows

Week 6: Webhooks and APIs

  • Webhooks and their purpose in Deluge
  • Sending data to external APIs using Deluge
  • Handling API responses
  • Integrating Deluge with third-party services
  • Review and practice with API integrations

Week 7: Advanced Deluge Applications

  • Working with files and attachments in Zoho
  • Creating reusable custom functions and libraries
  • Advanced date manipulations and scheduling tasks
  • Deluge and HTML/CSS integration
  • Sending automated emails with Deluge

Week 8: Special Topics and Best Practices

  • Implementing complex conditional logic in workflows
  • Best practices for writing clean and efficient code
  • Optimizing Deluge scripts for performance
  • Handling errors and logging issues effectively
  • Review and practice with special topics

Week 9: Capstone Project Introduction

  • Capstone project guidelines and expectations
  • Brainstorming and project planning
  • Identifying use cases for Deluge scripting
  • Review of learned concepts
  • Start working on the capstone project

Week 10: Capstone Project Development

  • Apply learned concepts to the capstone project
  • Work on refining the functionality of the project
  • Incorporate advanced features and optimizations
  • Prepare the final project for presentation
  • Review and reflect on the entire course experience
